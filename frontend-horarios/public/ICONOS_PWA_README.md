# Íconos PWA Pendientes

Para completar la configuración PWA, necesitas crear los siguientes íconos y colocarlos en esta carpeta (`public/`):

## Íconos Requeridos:

1. **pwa-192x192.png** (192 x 192 píxeles)
2. **pwa-512x512.png** (512 x 512 píxeles)

## Herramientas Sugeridas:

- PWA Builder Image Generator: https://www.pwabuilder.com/imageGenerator
- Real Favicon Generator: https://realfavicongenerator.net/
- Figma/Canva para diseñar el ícono base

## Recomendaciones:

- Usa el logo de tu sistema de horarios
- Los íconos deben ser cuadrados
- Usa fondo sólido o transparente
- El diseño debe ser simple y reconocible
- Colores sugeridos: Azul (#1890ff) según el tema de Ant Design

## Una vez creados los íconos:

1. Coloca `pwa-192x192.png` en esta carpeta
2. Coloca `pwa-512x512.png` en esta carpeta
3. Ejecuta `npm run build` para generar el build de producción
4. La aplicación será instalable como PWA

---

**Nota:** Mientras tanto, la aplicación funcionará normalmente, pero no será instalable hasta que los íconos estén presentes.
